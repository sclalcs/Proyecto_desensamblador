# Utilidad de Desensamblado

![Captura de la Utilidad](https://github.com/sclalcs/Proyecto_desensamblador/tree/main/Imagenes/)

## Descripción

Esta utilidad es un desensamblador desarrollado en **VB .NET 2008 Express Edition** utilizando la librería **SharpDisasm**. Permite cargar archivos binarios o secuencias de bytes y obtener su representación en código ensamblador de manera rápida y sencilla.

## Características

* Desensamblado utilizando **SharpDisasm**.
* Interfaz ligera y compatible con sistemas Windows antiguos.
* Soporte para múltiples arquitecturas dependiendo de la librería utilizada.
* Vista clara del código máquina y su equivalente en ensamblador.

## Requisitos

* **VB .NET 2008 Express Edition** o superior.
* **.NET Framework 3.5**.
* Librería **SharpDisasm** incluida en el proyecto.

## Instalación

1. Descargar o clonar este repositorio.
2. Abrir el proyecto con *VB .NET 2008 Express Edition*.
3. Restaurar la referencia a `SharpDisasm.dll` si fuera necesario.
4. Compilar y ejecutar.

## Uso

1. Abrir la utilidad.
2. Cargar un archivo binario o introducir bytes manualmente.
3. Seleccionar la arquitectura adecuada.
4. Visualizar el desensamblado generado.
